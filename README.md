# Slider de Carros

Este projeto é um slider de carros online que permite aos usuários navegar por uma galeria de imagens de carros.
![image](https://github.com/user-attachments/assets/56725a60-fa4e-4ce3-aa0a-5bda1008c3d7)

## Descrição

O slider de carros oferece uma interface simples e direta para visualizar imagens de carros. Os usuários podem navegar entre as imagens usando os botões "prev" e "next".

## Funcionalidades

* **Navegação por Imagens:** Navegue pelas imagens de carros usando os botões "prev" e "next".
* **JavaScript Interativo:** Lógica de navegação implementada em JavaScript para uma experiência dinâmica.
* **Estrutura HTML Organizada:** Estrutura HTML bem definida com classes e IDs para facilitar a manipulação via JavaScript.

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
